# Vigilante

**Vigilante** is a Python package designed for advanced web crawling, with special emphasis on anonymity and stealth capabilities. The flagship class, `Nightcrawler`, allows you to perform deep web searches with options to use the Tor network and operate in stealth mode.

## Features

- **Nightcrawler**: A class to initiate deep web crawls from a starting URL.
- **Tor Support**: Option to route crawling through the Tor network for anonymity.
- **Stealth Mode**: Minimalistic crawling to avoid detection.
- **Simple API**: Easy-to-use interface for both beginners and advanced users.
